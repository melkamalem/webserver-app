Key Projects

Microservices with Fargate to process event-tickets.

  • Deployed REST-API on API gateway to route requests to an SQS queue.
  • Created a DynamoDB table to store ticket information and built the Docker image of the application and pushed it to an ECR repository using the Cloud9 IDE
  • Created a CloudWatch alarm and ECS clusters with task definitions and services for the auto scaling.

Lyft and shift of shopping website to Serverless.

  • Created Aurora serverless cluster and migrated MySQL database to the cluster.
  • Configured Lambda function to deploy the application serverless.
  • Created an S3 bucket to host the application’s static files.
  • Configured a CloudFront distribution for the entry point of the application.
  • Created a custom metric filter using page-view logs on CloudWatch and created a lambda function for log processing.
  • Created a subscription filter for new subscriber and that invokes the lambda function.

Securing Networks and Ec2 services and monitoring overall security for upcoming migration to AWS.

  • Created a three-security zone network infrastructure.
  • Implemented network segmentation using security groups, network ACLs, and public and private subnets.
  • Monitored network traffic to EC2 instances using VPC flow logs.
  • Created a custom AMI and deployed a new EC2 instance from a custom AMI.
  • Patched an EC2 instance using AWS Systems Manager
  • Encrypted an EBS volume from a non-encrypted volume using snapshots
  • Configured an Amazon Linux 2 instance to send log files to Amazon CloudWatch

Built and Deployed a Containerized Application with Amazon Elastic Kubernetes Service
